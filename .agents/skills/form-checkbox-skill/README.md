# FormCheckbox Skill Project

Reusable skill package for implementing and validating `@rds-vue-ui/form-checkbox` in other projects.

## Source of Truth
- Storybook: `https://rds-vue-ui.edpl.us/?path=/story/components-forms-formcheckbox--primary`
- Docs ID: `components-forms-formcheckbox--docs`
- Story IDs: `components-forms-formcheckbox--primary`, `components-forms-formcheckbox--examples`
- NPM package: `@rds-vue-ui/form-checkbox` (`0.2.0`)
- Registry detail: `https://npm.edpl.us/-/web/detail/@rds-vue-ui/form-checkbox`

## Install
```bash
yarn add --registry=https://npm.edpl.us @rds-vue-ui/form-checkbox @rds-vue-ui/rds-theme-base
yarn add -D sass-embedded
```
