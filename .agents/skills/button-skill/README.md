# Button Skill Project (Base Theme Only)

This project documents the RDS `Button` component for the **base theme only**.

## What This Includes
- A practical component guide.
- A structured component spec.
- Starter examples for common usage.
- A QA checklist for doc validation.

## Theme Scope
- Included: base theme behavior and guidance.
- Excluded: all non-base themes and theme comparison notes.

## Audience
- Engineers implementing button interactions.
- Designers validating expected button behaviors.
- QA validating visual and accessibility behavior.

## Source of Truth
- RDS Storybook docs: `https://rds-vue-ui.edpl.us/?path=/docs/components-button-buttons--docs`

## Status
- Verification complete against Storybook compiled docs/stories assets.
- Current spec reflects class-based button usage documented for base theme.
